# Whats My Mods? 

A lightweight, high-visibility mod management tool for Gorilla Tag. This plugin allows you to see exactly what is running in your BepInEx environment without leaving the game.

## Features
* **Live Mod Count**: See exactly how many plugins are loaded.
* **In-Game Toggle**: Press `TAB` to show or hide the menu instantly.
* **Scrollable List**: Support for large mod folders.

## Installation
1. Ensure you have [BepInEx](https://github.com/BepInEx/BepInEx) installed.
2. Download the latest File from the **Releases** tab.
3. Drag the file into your `Gorilla Tag/BepInEx/plugins` folder.

## Controls
* **TAB**: Toggle the Ui.
